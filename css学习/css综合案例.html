段落1

  • 环境保护部18日通报，未来5天，京津冀中南部、山东西部和河南北部可能出现一次大范围持续重污染过程。北京市空气重污染应急指挥部发布今年第二次空气重污染红色预警：将于12月19日7时至12月22日24时实施空气重污染红色预警应急措施。
  • 据悉，北京、天津、石家庄、廊坊、唐山、保定、济南、淄博、郑州等地可能出现重度及以上污染，首要污染物为PM2.5，局地PM2.5峰值浓度将超过500微克/立方米。据中国环境监测总站预测，19日至20日，重污染影响范围将从河北中南部、山东西部、河南北部进一步往外扩展至山东大部和河南中部；21日至22日，重污染影响范围将继续维持，因湿度较大，污染程度进一步加重。据中央气象台预报，23日白天，受冷空气影响，雾霾将自北向南逐渐消散
  • 除京津冀及周边地区外，19日至21日，东北地区辽宁省中部个别城市可能达到重度污染；20日至22日，华中地区武汉至长沙沿线地区、陕西关中地区可能出现轻度到中度污染过程，局部地区可能短时出现重度污染；21日至22日，成渝地区可能出现轻度至中度污染，局部地区可能短时出现重度污染。
